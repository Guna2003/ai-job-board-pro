# 📖 Overview

AI Job Board is a modern frontend application designed to simulate a real-world job portal.

The application allows users to:

- Create an account
- Login securely
- Browse available jobs
- Search for jobs
- Apply to jobs
- View only their own applications
- Delete submitted applications

The project demonstrates modern React development practices, reusable component architecture, client-side routing, state management, and an automated CI/CD workflow using GitHub Actions and Vercel.



# 🚀 Deployment

This project is deployed using **Vercel**.

Every push to the **main** branch automatically triggers a new deployment after a successful GitHub Actions build.

---

# 🔄 CI/CD Pipeline

## Continuous Integration (CI)

GitHub Actions automatically performs:

- Repository Checkout
- Node.js Setup
- Dependency Installation
- Production Build Verification

This ensures every code push is validated before deployment.

## Continuous Deployment (CD)

Once the CI workflow succeeds:

GitHub → Vercel → Automatic Production Deployment

No manual deployment is required.
